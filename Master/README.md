# Accountability Module: Increasing Trust in Reinforcement Learning Agents

**MSc Thesis - University of Oslo, 2023**

An external explainability module for reinforcement learning agents. Built to answer why an autonomous vehicle crashed, using saliency maps, evolutionary algorithms, and UMAP counterfactuals to make black-box AI decisions transparent and auditable.

## Research Questions

- What were the main factors in the decision? (Saliency)
- Would changing a factor lead to a different decision? (Evolutionary Algorithm)
- Why did two similar cases get different decisions? (UMAP Counterfactual)

## Tech Stack

Python, PyTorch, stable-baselines3, DQN, highway-env, UMAP, Ray, CUDA
